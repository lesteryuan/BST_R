Running BST_R:

1. Edit "input choices.csv" to specify the scenario:
	siteid: "dry", "wet", "temperate"
	cas: CAS numbers of interest
	srctype: "LAU", "SI"
	srctypeid: "crop", "pasture", "reclamation"
	linertypeid: 0, 1, 2
2. Launch R.
3. source the script "main.R"
4. Model outputs are stored in "output"
